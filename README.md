# Trần Anh Đức — Premium Digital Portfolio V7

Astro one-page portfolio bằng tiếng Việt, thiết kế theo hướng blue liquid glass, docs-style, coding vibe và tối ưu trải nghiệm mượt.

## Chạy local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy Vercel

- Framework: Astro
- Build command: `npm run build`
- Output directory: `dist`

PDF gốc nằm trong `public/pdfs`.
